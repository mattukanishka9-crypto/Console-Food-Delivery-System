🍔 Console Food Delivery System (C++)

A simple console-based food delivery system developed in C++, demonstrating core programming concepts like conditional statements, loops, functions, and basic user interaction.

---

📌 Project Overview

This project simulates a basic food ordering system where users can:

- View available food items
- Select items from a menu
- Place an order
- View total bill

The entire application runs in the console and is implemented in a single source file: "Main.cpp".

---

⚙️ Features

- 📋 Display food menu with prices
- 🛒 Order multiple items
- ➕ Calculate total bill dynamically
- 🔁 Simple user interaction using loops
- ❌ Exit option for user convenience

---

🛠️ Technologies Used

- Language: C++
- Concepts:
  - Functions
  - Loops (for/while)
  - Conditional statements (if-else / switch)
  - Basic input/output

▶️ How to Run

1. Clone the repository:
   
   git clone https://github.com/your-username/your-repo-name.git

2. Navigate to the project folder:
   
   cd Console-Food-Delivery-System

3. Compile the program:
   
   g++ Main.cpp -o food_delivery

4. Run the executable:
   
   ./food_delivery

---

💡 Example Workflow

1. User starts the program
2. Menu is displayed
3. User selects food items by entering choices
4. System calculates total cost
5. Final bill is displayed
